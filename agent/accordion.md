## Task
Build an accordion using css, js, and html.

## CSS
- use BEM naming convention
- only style what is needed for functionality
## JS
use vanilla js

## HTML
use semantic html

## Accordion
- should have a title and a content
- should be able to open and close
- should have an icon that changes when the accordion is open or closed
- only one accordion should be open at a time
- first accordion should be open by default
- must be accessible
- use divs for the accordion and the content
